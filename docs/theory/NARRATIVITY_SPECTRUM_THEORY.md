## I. The Spectrum Defined

### Core Concept

**Narrativity**: The degree of narrative openness/constraint in a domain.

**Range**: 0.0 (totally circumscribed) to 1.0 (totally open)

### Examples Across Spectrum

| Domain | Narrativity | Description |
|--------|-------------|-------------|
| Die Roll | 0.12 | Physical laws determine outcome, no agency, instant |
| Basketball | 0.48 | Rules constrain, but agency within rules, temporal unfolding |
| Therapy | 0.61 | Open format, internal states, high interpretation |
| Diary | 0.61 | Creative freedom, participant narrative, flexible format |
| Startup Pitch | 0.76 | High apparent freedom, but market/format constraints |

## II. Five Components of Narrativity

### 1. Structural Openness (Weight: 0.30)

**How many possible narrative paths exist?**

- 0.0: Enumerable outcomes (die has 6 faces)
- 1.0: Infinite outcomes (diary can contain anything)

**Examples**:
- Die roll: 0.26 (6 discrete outcomes)
- Diary: 0.52 (unlimited content)

### 2. Temporal Freedom (Weight: 0.20)

**Can narrative unfold over time?**

- 0.0: Instant/single moment
- 1.0: Extended duration with development

**Examples**:
- Die roll: 0.00 (instant result)
- Basketball: 0.50 (48 minutes, momentum shifts)
- Diary: 0.00 (single entry) or 1.00 (lifelong journal)

### 3. Actor Agency (Weight: 0.25)

**How much choice do actors have?**

- 0.0: No agency (physics determines all)
- 1.0: Complete agency (can do anything)

**Examples**:
- Die roll: 0.00 (gravity decides)
- Basketball: 1.00 (players choose actions)
- Diary: 1.00 (writer chooses everything)

### 4. Observer Interpretation (Weight: 0.15)

**How much can interpretation vary?**

- 0.0: Objective (everyone sees same)
- 1.0: Subjective (each constructs different story)

**Examples**:
- Die roll: 0.25 (objective result, minimal interpretation)
- Basketball: 0.50 (objective score, subjective narratives about "why")
- Therapy: 1.00 (highly interpretive, meaning-making)

### 5. Format Flexibility (Weight: 0.10)

**How flexible is the medium/format?**

- 0.0: Rigid (die always has 6 sides)
- 1.0: Flexible (diary can be any form)

**Examples**:
- Die roll: 0.00 (fixed format)
- Startup pitch: 0.50 (deck structure but creative within)
- Diary: 0.50 (notebook constraints but flexible content)

## III. Narrator and Narrated

### Narrator Types

1. **Objective External**: No narrator (physics, math)
2. **Participant**: Actor narrates their own experience
3. **Observer**: Witness narrates what they see
4. **Omniscient**: God's-eye view of everything
5. **Collective**: Multiple narrators, ensemble view
6. **Algorithmic**: System generates narrative

### Narrated Types

1. **Physical Event**: Dice roll, chemical reaction
2. **Performance**: Game, execution, measurable action
3. **Interaction**: Relationship, conversation, connection
4. **Internal State**: Thoughts, feelings, perceptions
5. **Creation**: Making something new
6. **Trajectory**: Growth, journey, development over time

### Narrator-Narrated Coupling

**How tightly coupled?**

- 0.0: Separate (objective narrator of physical events)
- 1.0: Identical (participant narrating own internal state)

**Examples**:
- Die roll: 0.00 (objective→physical, completely separate)
- Basketball: 0.70 (participant→performance, tightly coupled)
- Therapy: 1.00 (participant→internal, identical)

## IV. Superficial vs Actual Potential

### Superficial Potential

**Apparent freedom before narrative unfolds.**

Many domains SEEM open but are actually constrained:
- Startup pitches SEEM flexible (high structural openness)
- But market-product fit constrains heavily
- Superficial = 0.60, Actual = 0.14
- **Gap = 0.46** (illusion of freedom)

### Actual Potential

**Real freedom after all constraints applied.**

What you CAN actually do within domain:
- Physical constraints (laws, biology)
- Format constraints (pitch deck, game rules)
- Temporal constraints (time limits)
- Social constraints (norms, expectations)
- Resource constraints (budget, capacity)

### The Gap (Illusion of Freedom)

**Superficial - Actual = Illusion**

Large gap means: Domain APPEARS open but is constrained.

**Examples**:
- Startup: Gap = 0.46 (seems open, actually constrained by market)
- Die roll: Gap = 0.13 (appears simple, is simple)
- Therapy: Gap = 0.30 (seems open, time/social constraints apply)

## V. The Key Relationship: Narrativity → α

### Empirical Finding

**Narrativity and α are inversely correlated: r=-0.961**

**Formula**: α ≈ 1.0 - Narrativity

**Interpretation**:
- Low narrativity (constrained) → High α (plot-driven)
- High narrativity (open) → Low α (character-driven)

**Why?**

When narrative is constrained (low narrativity):
- Few possible paths
- Outcomes determined by content/facts
- Plot dominates
- WHAT happens matters most

When narrative is open (high narrativity):
- Many possible paths
- Outcomes determined by choices/framing
- Character dominates
- WHO/HOW matters most

## X. Mathematical Formalization

### Narrativity Score

```
N = 0.30×S + 0.20×T + 0.25×A + 0.15×I + 0.10×F

Where:
S = Structural openness (0-1)
T = Temporal freedom (0-1)
A = Actor agency (0-1)
I = Observer interpretation (0-1)
F = Format flexibility (0-1)
```

### α Prediction

```
α = 1.0 - N + δ

Where:
δ = Adjustments based on:
    +0.2 if narrated = physical_event
    -0.2 if narrated = internal_state
    +0.15 if narrator = objective
    -0.10 if narrator = participant
```

### Superficial Potential

```
P_superficial = (S + F) / 2
```

### Actual Potential

```
P_actual = (DoF / 100) × (1 - mean(constraint_strengths))

Where DoF = Degrees of freedom
```

### Potential Gap (Illusion)

```
Gap = P_superficial - P_actual
```

**Large gap indicates**: Domain misleads about narrative freedom.
