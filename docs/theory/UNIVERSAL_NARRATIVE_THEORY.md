# Universal Narrative Theory: "Everything is a Book"

## The Meta-Theory

**Core Principle**: Every domain is a narrative (like a book) with the same structural elements, but domains differ in which elements drive outcomes.

**Not Metaphor**: Literal framework for analysis. All domains genuinely have:
- Characters/entities with names and identities
- Relationships/networks (ensemble)
- Communication style (voice)
- Content/plot (what's described)
- Development arcs (trajectories)
- Context (genre, stage, comparisons)

**The Discovery**: Testing reveals which elements matter WHERE.

---

## The Universal Elements (Present Everywhere)

### 1. Names/Identity (Nominative Dimension)

**In Books**: Character names carry meaning, positioning  
**In Domains**:
- MMA: Fighter names signal identity (harsh = dangerous)
- Crypto: Project names signal positioning (connects to Bitcoin)
- Brands: Company names signal values
- People: Personal names signal background

**When Matters**: Identity-driven domains (MMA, personal profiles, branding)  
**Evidence**: MMA shows r=0.568 between name harshness and performance

### 2. Character Development (Self-Perception + Potential Dimensions)

**In Books**: Characters grow, change, have arcs  
**In Domains**:
- Wellness: Growth mindset in journals
- Therapy: Development trajectory
- Careers: Professional growth narrative
- Crypto: Project roadmap/vision

**When Matters**: Growth-focused domains (wellness, education, forecasting)  
**Evidence**: Crypto Potential transformer: 93.7% (future-vision matters)

### 3. Ensemble/Relationships (Ensemble + Relational Dimensions)

**In Books**: Cast dynamics, relationships between characters  
**In Domains**:
- Relationships: How interests/values connect
- Crypto: Ecosystem positioning (connects to majors)
- Teams: Member dynamics
- Networks: Connection patterns

**When Matters**: Relational domains (dating, teams, social)  
**Evidence**: Crypto Ensemble: 93.8% (ecosystem positioning crucial)

### 4. Voice/Style (Linguistic Dimension)

**In Books**: How story is told (voice, emotion, language)  
**In Domains**:
- Reviews: Emotional language, personal voice
- Opinion: How you express matters
- Content: Engaging vs boring style
- Communication: Professional vs casual voice

**When Matters**: Opinion/communication domains (reviews, content, messaging)  
**Evidence**: Linguistic 37% on news but expected 78%+ on IMDB reviews

### 5. Plot/Content (Statistical Dimension)

**In Books**: What happens, events, actions  
**In Domains**:
- News: What happened (content IS the story)
- Technical: What it does
- Instructions: What steps
- Reports: What occurred

**When Matters**: Content-focused domains (news, technical, reports)  
**Evidence**: News Statistical 69% vs Ensemble 28% (plot dominates)

### 6. Arc/Context (Temporal + Contextual Dimensions)

**In Books**: Where in story, genre expectations, series position  
**In Domains**:
- MMA: Weight class, career stage (like character in different parts of saga)
- Tennis: Surface type (like different romance sub-genres)
- Relationships: Relationship stage (like story act)
- Teams: Season position (like plot progression)

**When Matters**: Always (moderates other effects)  
**Evidence**: MMA heavyweight r=0.628 vs lightweight r=0.544 (same dimension, different context)
