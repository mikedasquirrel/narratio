statistical methodology

All systems tested on holdout data with proper temporal splits. Meta-Ensemble combines Random Forest, Gradient Boosting, and Logistic Regression. Features extracted via universal transformer pipeline (79 dimensions for NHL). Confidence thresholds calibrated on training data, applied to unseen test data. Binomial tests confirm statistical significance (p < 0.001 for NHL).
