## 1. Chinese Narrative Theory

### Theoretical Foundation

**Core Concepts**:

1. **Qi (氣) - Vital Energy Flow**
   - Narrative momentum as energy circulation
   - Must flow smoothly without blockages
   - Maps to temporal dynamics (ρ, flow continuity)

2. **Yin-Yang (陰陽) - Binary Oppositions**
   - Every narrative balances opposites
   - Light/dark, active/passive, rising/falling
   - Maps to conflict structure

3. **Five Elements (五行) - 5-Act Structure**
   - Wood (木): Birth, spring, beginning → Act 1
   - Fire (火): Growth, summer, expansion → Act 2
   - Earth (土): Transformation, center, climax → Act 3
   - Metal (金): Harvest, autumn, resolution → Act 4
   - Water (水): Storage, winter, return → Act 5

4. **Confucian Harmony (和)**
   - Resolution requires social balance
   - Individual desires subordinate to harmony
   - Contrasts with Western individual hero

5. **Cyclical Time**
   - History repeats in cycles
   - Dynasties rise and fall
   - Eternal recurrence vs linear progress

### Mathematical Mapping

```python
# Qi maps to ρ (rhythm) + momentum
Qi_flow = 1 / (1 + ρ)  # Smooth flow = low rhythm variation
blockage_penalty = abs(momentum_acceleration)  # Sudden stops

# Yin-Yang maps to conflict balance
yin_yang_balance = 1 - abs(positive_force - negative_force)
# Perfect balance = 1, imbalance = 0

# Five Elements map to act structure
element_progression = [
    (0.00, 0.20, 'wood'),   # Setup
    (0.20, 0.40, 'fire'),   # Rising action
    (0.40, 0.60, 'earth'),  # Climax/transformation
    (0.60, 0.80, 'metal'),  # Falling action
    (0.80, 1.00, 'water')   # Resolution
]

# Check if narrative follows element progression
element_adherence = measure_progression_quality(beats, element_progression)
```

### Features to Extract (15 features)

1. Qi flow quality (1 - blockage_count)
2. Yin-yang balance score
3. Five element progression adherence
4. Confucian harmony markers (group > individual)
5. Cyclical pattern recognition
6. Seasonal metaphor use
7. Nature-human integration
8. Social hierarchy respect
9. Elder wisdom inclusion
10. Filial piety markers
11. Balance restoration (harmony achieved)
12. Transformation through center (earth element emphasis)
13. Water return (cyclical completion)
14. Energy culmination (fire peak)
15. Generational continuity

### Validation Hypothesis

## 2. Japanese Narrative Theory

### Theoretical Foundation

**Core Concepts**:

1. **Mono no Aware (物の哀れ) - Pathos of Things**
   - Beauty in transience and impermanence
   - Awareness of mortality/ephemerality
   - Contrasts with Western emphasis on permanence/triumph

2. **Ma (間) - Negative Space**
   - Silence is as important as sound
   - Pauses are part of rhythm
   - What's NOT said matters
   - Maps to breathing room, pause density

3. **Kata (型) - Fixed Forms**
   - Ritualized structures (Noh, Kabuki)
   - Repetition as aesthetic choice
   - Form mastery before innovation

4. **Jo-ha-kyū (序破急) - 3-Part Acceleration**
   - Jo (序): Slow introduction, establishment
   - Ha (破): Break, acceleration, development
   - Kyū (急): Rapid, climax, completion
   - Maps to ρ (rhythm) with specific acceleration curve

5. **Okashi vs Aware**
   - Okashi: Charm, humor, brightness (Heian period)
   - Aware: Deep feeling, melancholy, transience (Later aesthetic)
   - Different emotional registers

### Mathematical Mapping

```python
# Mono no aware: Impermanence emphasis
impermanence_score = count_transience_markers(text)
# Markers: "fleeting", "moment", "passing", "ephemeral"

# Ma: Pause density and quality

# Pauses should come at natural boundaries, not arbitrarily

# Jo-ha-kyū: Specific acceleration pattern

kyu_duration = 0.20  # Rapid conclusion

jo_ha_kyu_adherence = measure_acceleration_pattern(beats, [jo, ha, kyu])

# Acceleration should be: slow → moderate → rapid
# Measured by ς (compression) increase over time
ideal_acceleration = [0.5, 1.0, 2.0]  # Doubling each phase
```

### Features to Extract (15 features)

1. Impermanence theme strength
2. Transience acceptance (vs resistance)
3. Ma (negative space) quality
4. Pause appropriateness
5. Silence integration
6. Jo-ha-kyū adherence
7. Acceleration pattern match
8. Kata (form) mastery indicators
9. Repetition as beauty (not flaw)
10. Seasonal imagery (cherry blossoms, autumn leaves)
11. Melancholy tone (aware aesthetic)
12. Restraint markers (understatement)
13. Subtlety score
14. Natural imagery integration
15. Zen influence markers (sudden enlightenment)

### Validation Hypothesis

## 3. Indian Rasa Theory (रस)

### Theoretical Foundation

**Core Concepts**:

**9 Rasas (Emotional Essences)**:
1. **Śṛṅgāra (शृङ्गार)**: Love, beauty, attraction
2. **Hāsya (हास्य)**: Mirth, comedy, laughter
3. **Karuṇā (करुणा)**: Compassion, pathos, sadness
4. **Raudra (रौद्र)**: Fury, anger, rage
5. **Vīra (वीर)**: Heroic, courage, valor
6. **Bhayānaka (भयानक)**: Terror, fear, horror
7. **Bībhatsa (बीभत्स)**: Disgust, aversion
8. **Adbhūta (अद्भुत)**: Wonder, amazement
9. **Śānta (शान्त)**: Peace, tranquility (later addition)

**Key Principles**:
- Narrative as emotional journey through rasas
- Not plot-driven, but emotion-driven
- Audience should experience (not observe) rasas
- Rasa progression has patterns

**Bhāvas (States)**: Emotions arise from combinations of:
- Vibhāva (causes): Situations that evoke
- Anubhāva (effects): Expressions manifesting
- Vyabhicāri (transitory): Passing feelings

**Cyclical Time**: Karma, rebirth affect narrative structure differently than linear Western time

### Mathematical Mapping

```python
# Rasa distribution across narrative

    raudra_intensity,    # Fury

    bhayanaka_intensity, # Terror
    bibhatsa_intensity,  # Disgust

    shanta_intensity     # Peace

# Rasa dominance (which rasa is primary?)

# Rasa balance (is one over-represented?)

# High entropy = balanced rasas, low = single rasa dominant

# Rasa progression quality
# Traditional patterns: Start low → Build → Resolve
# E.g.: Śānta → Vīra → Raudra → Karuṇā → Śānta (peace to peace)

# Integration with Western model
# Rasas are MORE FUNDAMENTAL than Western emotions
# Map Western emotions → Rasas, then analyze
```

### Features to Extract (18 features)

One intensity score per rasa (9) + 9 structural:
1-9. Intensity of each rasa

13. Śānta bookends (peace beginning & end)
14. Emotional complexity (number of rasas present)
15. Rasa transitions (abrupt vs smooth)

   - Language as art form

3. **Trickster Centrality**
   - Rogue protagonist (al-Hariri's Abu Zayd)
   - Not heroic journey but picaresque
   - Wisdom through cunning, not virtue

4. **Frame Narrative**
   - Story within story structure
   - Narrator distance (hikāya)
   - Multiple layers of narrative

```python
        ]
    
    def transform(self, X, metadata=None):
        features_list = []
        
        for narrative in X:
            # Test fit to each tradition
            fit_scores = {}
            for tradition in self.traditions:
                fit_scores[tradition.name] = tradition.measure_fit(narrative)
            
            # Best-fit tradition
            best_tradition = max(fit_scores, key=fit_scores.get)
            
            # Extract features from all traditions
            all_features = []
            for tradition in self.traditions:
                tradition_features = tradition.extract_features(narrative)
                all_features.extend(tradition_features)
            
            # Add meta-features
            all_features.append(fit_scores[best_tradition])  # Best fit score
            all_features.append(self._measure_hybridity(fit_scores))  # Is it hybrid?
            all_features.append(self._extract_universal_core(narrative))  # Universal elements
            all_features.append(self._measure_cultural_distance(fit_scores))  # How distinct?
            
            features_list.append(all_features)
        
        return np.array(features_list)
```
