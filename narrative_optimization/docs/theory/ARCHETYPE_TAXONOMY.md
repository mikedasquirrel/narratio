# Complete Archetype Taxonomy

**Version**: 1.0.0  
**Date**: November 13, 2025  
**Purpose**: Hierarchical classification system for all narrative archetypes across character, plot, theme, and structure

---

## Overview

This document provides a comprehensive taxonomic organization of narrative archetypes from all major theories (Jung, Campbell, Vogler, Propp, Booker, Frye, Aristotle). The taxonomy is hierarchical, machine-readable, and integrates seamlessly with the π/λ/θ/ة framework.

---

## Taxonomy Structure

```
ARCHETYPES
├── CHARACTER ARCHETYPES
│   ├── Psychological (Jung/Vogler)

├── PLOT ARCHETYPES

```

## 1. CHARACTER ARCHETYPES

### 1.1 Psychological Archetypes (Jung + Vogler)

#### 1.1.1 The Hero Family

**Primary Hero**: Protagonist on transformative journey
- **Willing Hero**: Embraces call immediately
  - Markers: `['eager', 'ready', 'destined', 'chosen']`
  - Agency: 1.0 (maximum)
  - Example: Luke Skywalker (Star Wars)
  
- **Unwilling Hero**: Reluctant, forced into action
  - Markers: `['reluctant', 'forced', 'no choice', 'must']`
  - Agency: 0.7 (constrained)
  - Example: Frodo (Lord of the Rings)
  
- **Anti-Hero**: Morally ambiguous protagonist
  - Markers: `['flawed', 'morally grey', 'questionable', 'dark']`
  - Agency: 0.8
  - Example: Walter White (Breaking Bad)
  
- **Tragic Hero**: Doomed by fatal flaw
  - Markers: `['doomed', 'hubris', 'fatal flaw', 'downfall']`
  - Agency: 0.6 (fate-constrained)
  - Example: Oedipus, Macbeth

**Group Hero**: Collective protagonist
  - Markers: `['team', 'together', 'united', 'ensemble']`
  - Agency: 0.5 (distributed)
  - Example: Fellowship, Avengers

#### 1.1.2 The Mentor Family

**Wise Mentor**: Traditional teacher/guide
- Markers: `['wisdom', 'guide', 'teach', 'elder', 'master']`
- Function: Prepare hero, provide tools
- Example: Gandalf, Obi-Wan, Dumbledore

**Dark Mentor**: Teaches wrong lessons
- Markers: `['corrupt', 'mislead', 'false wisdom', 'dangerous']`
- Function: Lead hero astray
- Example: Palpatine, Tyler Durden

**Fallen Mentor**: Once wise, now compromised
- Markers: `['fallen', 'corrupted', 'lost way', 'broken']`
- Function: Cautionary tale
- Example: Saruman, Harvey Dent

**Absent Mentor**: Dead/unavailable, legacy guides
- Markers: `['memory', 'legacy', 'lost', 'ghost']`
- Function: Inspiration through absence
- Example: Uncle Ben, Mufasa (ghost)

**Shaman Mentor**: Spiritual/mystical guide
- Markers: `['mystical', 'spiritual', 'vision', 'prophecy']`
- Function: Spiritual awakening
- Example: Morpheus, Yoda

#### 1.1.3 The Shadow Family

**External Shadow**: Literal antagonist
- Markers: `['villain', 'enemy', 'antagonist', 'oppose']`
- Function: Physical/external obstacle
- Example: Darth Vader, Voldemort

**Internal Shadow**: Protagonist's dark side
- Markers: `['inner demon', 'temptation', 'weakness', 'darkness within']`
- Function: Psychological obstacle
- Example: Jekyll/Hyde duality

**Sympathetic Shadow**: Understandable villain
- Markers: `['tragic', 'misunderstood', 'justified', 'complex']`
- Function: Moral complexity
- Example: Magneto, Killmonger

**Redeemable Shadow**: Can be saved
- Markers: `['redemption', 'saved', 'good inside', 'turn back']`
- Function: Redemption arc
- Example: Darth Vader (return), Loki

**Pure Evil Shadow**: Irredeemable darkness
- Markers: `['evil', 'corrupt', 'soulless', 'irredeemable']`
- Function: Absolute opposition
- Example: Sauron, Emperor Palpatine

#### 1.1.4 Supporting Archetypes

**Threshold Guardian**: Tests commitment
- Markers: `['test', 'prove', 'worthy', 'block', 'challenge']`
- Function: Gate-keeping, testing
- Example: Bouncers, bureaucrats, initial obstacles

**Herald**: Announces change
- Markers: `['message', 'announce', 'summons', 'call', 'news']`
- Function: Deliver call to adventure
- Example: R2-D2 with message, messenger characters

**Shapeshifter**: Creates doubt
- Markers: `['ambiguous', 'unclear', 'mysterious', 'unpredictable']`
- Function: Suspense, uncertainty
- Example: Severus Snape, femme fatales

**Trickster**: Comic relief, chaos
- Markers: `['mischief', 'chaos', 'humor', 'disrupt', 'playful']`
- Function: Break tension, catalyze change
- Example: Loki, Puck, Donkey (Shrek)

**Ally**: Faithful companion
- Markers: `['loyal', 'friend', 'companion', 'sidekick', 'support']`
- Function: Support hero
- Example: Samwise, Ron/Hermione, Robin

#### 1.1.5 Jung's 12 Primary Archetypes

**Ego Types** (seeking paradise):

**1. The Innocent**
- Core desire: Safety, happiness
- Fear: Punishment, abandonment
- Strategy: Faith, optimism
- Markers: `['innocent', 'pure', 'naive', 'optimistic', 'trusting']`
- Example: Dorothy (Oz), Forrest Gump

**2. The Orphan/Everyman**
- Core desire: Connection, belonging
- Fear: Being left out
- Strategy: Realism, common touch
- Markers: `['ordinary', 'relatable', 'common', 'everyman', 'realistic']`
- Example: Harry Potter (early), Bilbo

**3. The Warrior/Hero**
- Core desire: Prove worth
- Fear: Weakness, cowardice
- Strategy: Courage, discipline
- Markers: `['brave', 'courageous', 'fight', 'protect', 'sacrifice']`
- Example: Achilles, Wonder Woman

**4. The Caregiver**
- Core desire: Help others
- Fear: Selfishness, ingratitude
- Strategy: Compassion, generosity
- Markers: `['nurture', 'protect', 'selfless', 'compassion', 'care']`
- Example: Samwise, Mary Poppins

**Soul Types** (seeking connection):

**5. The Explorer**
- Core desire: Freedom, discovery
- Fear: Conformity, emptiness
- Strategy: Journey, autonomy
- Markers: `['freedom', 'explore', 'discover', 'independent', 'wander']`
- Example: Huck Finn, Indiana Jones

**6. The Rebel/Outlaw**
- Core desire: Revolution, change
- Fear: Powerlessness
- Strategy: Disruption, breaking rules
- Markers: `['rebel', 'revolution', 'break rules', 'disrupt', 'overthrow']`
- Example: Robin Hood, V (V for Vendetta)

**7. The Lover**
- Core desire: Intimacy, connection
- Fear: Being alone, unwanted
- Strategy: Passion, commitment
- Markers: `['love', 'passion', 'intimate', 'devoted', 'romantic']`
- Example: Romeo & Juliet, Rose (Titanic)

**8. The Creator**
- Core desire: Create enduring value
- Fear: Mediocrity, stagnation
- Strategy: Innovation, imagination
- Markers: `['create', 'innovate', 'imagine', 'artist', 'build']`
- Example: Willy Wonka, Tony Stark

**Self Types** (seeking order):

**9. The Jester**
- Core desire: Joy, living in moment
- Fear: Boredom
- Strategy: Play, humor
- Markers: `['humor', 'playful', 'joke', 'lighthearted', 'fun']`
- Example: Puck, Ferris Bueller

**10. The Sage**
- Core desire: Truth, understanding
- Fear: Ignorance, deception
- Strategy: Knowledge, analysis
- Markers: `['wise', 'knowledge', 'truth', 'analyze', 'understand']`
- Example: Gandalf, Dumbledore

**11. The Magician**
- Core desire: Transform reality
- Fear: Unintended consequences
- Strategy: Vision, transformation
- Markers: `['magic', 'transform', 'vision', 'power', 'mystical']`
- Example: Merlin, Morpheus

**12. The Ruler**
- Core desire: Control, order
- Fear: Chaos, overthrow
- Strategy: Leadership, responsibility
- Markers: `['lead', 'control', 'order', 'command', 'authority']`
- Example: King Arthur, Black Panther

### 1.2 Functional Archetypes (Propp's 7 Spheres)

**Villain**
- Function: Struggles against hero
- Actions: Creates conflict, harm, obstacle
- Markers: `['antagonist', 'villain', 'oppose', 'harm', 'destroy']`

**Donor**
- Function: Provides magical agent/tool
- Actions: Tests hero, gives item/knowledge
- Markers: `['give', 'provide', 'gift', 'grant', 'bestow']`

**Helper**
- Function: Aids hero in quest
- Actions: Assists, rescues, supports
- Markers: `['help', 'assist', 'aid', 'support', 'rescue']`

**Princess (Sought-for-Person)**
- Function: Goal of quest, marries hero
- Actions: Rescued, won, united with
- Markers: `['sought', 'rescued', 'prize', 'goal', 'beloved']`

**Dispatcher**
- Function: Sends hero on quest
- Actions: Commands, requests, dispatches
- Markers: `['send', 'dispatch', 'command', 'request', 'summon']`

**Hero**
- Function: Defeats villain, achieves goal
- Actions: Quests, battles, succeeds
- Markers: `['quest', 'seek', 'battle', 'achieve', 'triumph']`

**False Hero**
- Function: Claims hero's credit falsely
- Actions: Lies, deceives, claims credit
- Markers: `['false', 'imposter', 'claim', 'deceive', 'pretend']`

### 1.3 Role-Based Archetypes (Traditional)

**Protagonist**: Main character
**Antagonist**: Opposition
**Deuteragonist**: Second most important
**Tritagonist**: Third importance
**Confidant**: Character hero trusts
**Foil**: Contrasts with protagonist
**Love Interest**: Romantic object
**Comic Relief**: Provides humor

## 2. PLOT ARCHETYPES

### 2.1 Master Plots (Booker's Seven)

#### 2.1.1 Overcoming the Monster

**Structure**: 5 stages
1. **Anticipation Stage**: Monster threatens
   - Markers: `['threat', 'danger', 'monster', 'evil approaching']`
2. **Dream Stage**: Hero prepares, initial confidence
   - Markers: `['prepare', 'train', 'ready', 'confident']`
3. **Frustration Stage**: First battles difficult
   - Markers: `['difficult', 'struggle', 'harder than expected']`
4. **Nightmare Stage**: Almost defeated
   - Markers: `['desperate', 'nearly defeated', 'all seems lost']`
5. **Miraculous Escape**: Hero triumphs
   - Markers: `['triumph', 'victory', 'defeated monster', 'saved']`

**Requirements**:
- Clear monster/threat (literal or metaphorical)
- Escalating danger
- Final confrontation
- Victory

**Examples**: Beowulf, Jaws, Star Wars, Dracula

#### 2.1.2 Rags to Riches

**Structure**: 5 stages
1. **Initial Wretchedness**: Hero poor/oppressed
   - Markers: `['poor', 'oppressed', 'lowly', 'nothing', 'despised']`
2. **Out Into World**: Gets opportunity
   - Markers: `['opportunity', 'chance', 'possibility', 'opens up']`
3. **Initial Success**: First taste of better life
   - Markers: `['success', 'better', 'improved', 'rise', 'advance']`
4. **Central Crisis**: Threatened with loss
   - Markers: `['threatened', 'might lose', 'crisis', 'danger']`
5. **Independence**: Proves true worth
   - Markers: `['proved', 'worthy', 'deserving', 'earned', 'achieved']`

**Examples**: Cinderella, Oliver Twist, Pretty Woman, Slumdog Millionaire

#### 2.1.3 The Quest

**Structure**: 5 stages
1. **The Call**: Learn of goal
   - Markers: `['quest', 'seek', 'find', 'journey', 'goal']`
2. **The Journey**: Travel with companions
   - Markers: `['travel', 'journey', 'companions', 'together', 'onward']`
3. **Arrival and Frustration**: Reach location, obstacles
   - Markers: `['arrived', 'reached', 'but', 'obstacle', 'blocked']`
4. **Final Ordeal**: Ultimate test
   - Markers: `['ultimate', 'final test', 'greatest challenge', 'ordeal']`
5. **The Goal**: Achievement, return
   - Markers: `['achieved', 'found', 'attained', 'succeeded', 'return']`

**Examples**: The Odyssey, Lord of the Rings, Raiders of the Lost Ark

#### 2.1.4 Voyage and Return

**Structure**: 5 stages
1. **Anticipation and Fall**: Enter strange world
   - Markers: `['strange', 'new world', 'different', 'unknown', 'entered']`
2. **Initial Fascination**: Wonderful exploration
   - Markers: `['wonder', 'amazing', 'fascination', 'explore', 'magical']`
3. **Frustration**: Threat emerges
   - Markers: `['threat', 'danger', 'turns dark', 'sinister', 'trapped']`
4. **Nightmare**: Everything goes wrong
   - Markers: `['nightmare', 'horror', 'terror', 'desperate', 'trapped']`
5. **Thrilling Escape**: Return home
   - Markers: `['escape', 'return', 'home', 'safe', 'back']`

**Examples**: Alice in Wonderland, The Wizard of Oz, Narnia, Spirited Away

#### 2.1.5 Comedy

**Structure**: 5 stages
1. **Establishment**: Normal world, brewing confusion
   - Markers: `['established', 'normal', 'but', 'confusion', 'mistake']`
2. **Complication**: Misunderstandings multiply
   - Markers: `['complication', 'misunderstanding', 'confused', 'tangle']`
3. **Complete Confusion**: Chaos, identities mixed
   - Markers: `['chaos', 'complete confusion', 'everyone thinks', 'mistaken']`
4. **Resolution**: Truth revealed
   - Markers: `['revealed', 'truth', 'realize', 'understand', 'clear']`
5. **Union**: Community restored, marriage
   - Markers: `['married', 'united', 'together', 'harmony', 'celebration']`

**Examples**: Shakespeare's comedies, Pride and Prejudice, When Harry Met Sally

#### 2.1.6 Tragedy

**Structure**: 5 stages
1. **Anticipation Stage**: Hero incomplete, desires
   - Markers: `['desire', 'want', 'incomplete', 'ambitious', 'seeking']`
2. **Dream Stage**: Initial success
   - Markers: `['success', 'achievement', 'power', 'attained', 'triumphant']`
3. **Frustration Stage**: Opposition appears
   - Markers: `['opposition', 'threat', 'challenged', 'questioned']`
4. **Nightmare Stage**: Everything falls apart
   - Markers: `['falling apart', 'losing', 'destruction', 'doom', 'inevitable']`
5. **Destruction or Death**: Final catastrophe
   - Markers: `['death', 'destruction', 'fallen', 'ruined', 'ended']`

**Examples**: Macbeth, Anna Karenina, Breaking Bad, Hamlet

#### 2.1.7 Rebirth

**Structure**: 5 stages
1. **Fall Under Shadow**: Dark spell begins
   - Markers: `['cursed', 'spell', 'enchanted', 'transformed', 'trapped']`
2. **Recession**: Pressure seems to ease
   - Markers: `['seems better', 'temporary relief', 'false hope']`
3. **Imprisonment**: Fully trapped in darkness
   - Markers: `['trapped', 'imprisoned', 'lost', 'darkness', 'no escape']`
4. **Nightmare**: Total darkness
   - Markers: `['darkest', 'hopeless', 'despair', 'end', 'no way']`
5. **Redemption**: Transformation, rebirth
   - Markers: `['reborn', 'transformed', 'redeemed', 'saved', 'awakened']`

**Examples**: Beauty and the Beast, A Christmas Carol, Groundhog Day

## 3. THEMATIC ARCHETYPES

### 3.1 Mythoi (Frye's Four Seasons)

#### 3.1.1 Comedy (Spring)

**Characteristics**:
- Season: Spring
- Movement: Integration, renewal
- Society: Young overcome old blocking figures
- Ending: Marriage, feast, new order
- Mood: Optimistic, celebratory

**Sub-types**:
- **Romantic Comedy**: Love obstacles overcome
- **Satirical Comedy**: Social folly mocked
- **Farce**: Physical/situational humor
- **Comedy of Manners**: Social customs satirized
- **Dark Comedy**: Serious undertones

**Markers**:
```python
comedy_markers = [
    'marry', 'wedding', 'united', 'together',
    'celebration', 'feast', 'harmony', 'resolved',
    'misunderstanding cleared', 'revealed', 'happy ending'
]
```

#### 3.1.2 Romance (Summer)

**Characteristics**:
- Season: Summer
- Movement: Adventure, idealization
- Structure: Quest, battle good vs evil
- Setting: Exotic, magical
- Mood: Optimistic, heroic

**Sub-types**:
- **Quest Romance**: Search for object/person
- **Adventure Romance**: Journey-focused
- **Chivalric Romance**: Knights, honor codes
- **Gothic Romance**: Dark, mysterious settings
- **Fantasy Romance**: Magical elements

**Markers**:
```python
romance_markers = [
    'quest', 'adventure', 'journey', 'hero',
    'noble', 'brave', 'good vs evil', 'magical',
    'exotic', 'triumph', 'victory', 'glory'
]
```

#### 3.1.3 Tragedy (Autumn)

**Characteristics**:
- Season: Autumn
- Movement: Fall, decline, sacrifice
- Protagonist: Noble with fatal flaw
- Structure: Rise to peak, then fall
- Mood: Serious, inevitable

**Sub-types**:
- **Classical Tragedy**: Greek/Shakespearean
- **Domestic Tragedy**: Common people
- **Heroic Tragedy**: Noble protagonists
- **Revenge Tragedy**: Vengeance-driven
- **Modern Tragedy**: Existential themes

**Markers**:
```python
tragedy_markers = [
    'fall', 'downfall', 'hubris', 'pride', 'flaw',
    'inevitable', 'fate', 'doom', 'death', 'sacrifice',
    'too late', 'realized', 'ruined', 'catastrophe'
]
```

#### 3.1.4 Irony/Satire (Winter)

**Characteristics**:
- Season: Winter
- Movement: Decline, disillusionment
- Protagonist: Unheroic, passive
- Society: System dominates individual
- Mood: Realistic, pessimistic, ambiguous

**Sub-types**:
- **Satire**: Social/political critique
- **Irony**: Expectations vs reality
- **Nihilism**: Meaninglessness
- **Absurdism**: Logical impossibility
- **Dystopia**: Nightmarish society

**Markers**:
```python
irony_markers = [
    'futile', 'meaningless', 'absurd', 'no point',
    'system', 'trapped', 'powerless', 'ambiguous',
    'unclear', 'unresolved', 'cycle', 'again'
]
```

**Moral Ambiguity**: Gray areas
**Redemption**: Possible transformation
**Damnation**: Irredeemable fall
**Justice**: Balance restored
**Injustice**: Unfairness prevails
**Sacrifice**: Greater good trade-off
**Corruption**: Moral degradation

**Existential**: Meaning-making, choice, freedom
**Nihilistic**: Meaninglessness, void
**Humanistic**: Human potential, growth
**Deterministic**: Fate, predestination
**Absurdist**: Logical impossibility
**Optimistic**: Hope, progress
**Pessimistic**: Decline, darkness
**Dualistic**: Two opposing forces

## 4. STRUCTURAL ARCHETYPES

### 4.1 Act Structures

#### 4.1.1 Three-Act Structure (Aristotle/Field)

**Act I: Setup** (25%)
- Exposition
- Inciting incident (10-15%)
- Plot Point 1 (25%)

**Act II: Confrontation** (50%)
- Rising action
- Midpoint (50%)
- Plot Point 2 (75%)

**Act III: Resolution** (25%)
- Climax
- Falling action
- Resolution

#### 4.1.2 Five-Act Structure (Shakespeare)

**Act I**: Exposition
**Act II**: Rising Action
**Act III**: Climax
**Act IV**: Falling Action
**Act V**: Denouement

#### 4.1.3 Seven-Act Structure (Hollywood)

1. Hook
2. Plot Point 1
3. Pinch 1
4. Midpoint
5. Pinch 2
6. Plot Point 2
7. Resolution

### 4.2 Beat Patterns (Blake Snyder's Save the Cat)

**15 Beats**:
1. Opening Image (1%)
2. Theme Stated (5%)
3. Setup (1-10%)
4. Catalyst (12%)
5. Debate (12-25%)
6. Break into Two (25%)
7. B Story (30%)
8. Fun and Games (30-55%)
9. Midpoint (50%)
10. Bad Guys Close In (55-75%)
11. All Is Lost (75%)
12. Dark Night of the Soul (75-85%)
13. Break into Three (85%)
14. Finale (85-99%)
15. Final Image (100%)

### 4.3 Temporal Shapes (Vonnegut)

**8 Story Shapes**:
1. **Man in Hole**: Neutral → Down → Up
2. **Boy Meets Girl**: Down → Up → Down → Up higher
3. **From Bad to Worse**: Steady decline
4. **Which Way Is Up?**: Ambiguous/chaotic
5. **Creation Story**: Exponential rise
6. **Old Testament**: Up → Down → Partial redemption
7. **New Testament**: Low → Sacrifice → Eternal high
8. **Cinderella**: Low → High → Low → Higher

## 6. COMPUTATIONAL REPRESENTATION

### 6.1 Archetype Feature Vector

```python
class ArchetypeProfile:
    """
    Complete archetype representation for a narrative
    """
    def __init__(self):
        # Character archetypes (vectorized)
        self.jung_archetypes = np.zeros(12)  # 12 Jung archetypes
        self.vogler_archetypes = np.zeros(8)  # 8 Vogler roles
        self.propp_spheres = np.zeros(7)  # 7 Propp spheres
        
        # Plot archetypes
        self.booker_plots = np.zeros(7)  # 7 master plots
        self.campbell_stages = np.zeros(17)  # 17 hero journey stages
        self.polti_situations = np.zeros(36)  # 36 dramatic situations
        
        # Thematic archetypes
        self.frye_mythoi = np.zeros(4)  # 4 mythoi
        self.moral_framework = ""  # Text label
        self.philosophical_pattern = ""  # Text label
        
        # Structural archetypes
        self.act_structure = ""  # '3-act', '5-act', etc.
        self.beat_pattern = np.zeros(15)  # Save the Cat beats
        self.vonnegut_shape = ""  # Story shape label
        
    def to_vector(self):
        """Flatten to single feature vector"""
        return np.concatenate([
            self.jung_archetypes,
            self.vogler_archetypes,
            self.propp_spheres,
            self.booker_plots,
            self.campbell_stages,
            self.polti_situations,
            self.frye_mythoi,
            self.beat_pattern
        ])  # Total: 131 numerical features
```

def extract_complete_archetype_profile(narrative_data):
    """
    Complete archetype extraction
    
    Args:
        narrative_data: {
            'full_text': complete narrative,
            'character_texts': {name: text for each character},
            'metadata': additional info
        }
    
    Returns:
        Complete ArchetypeProfile
    """
    profile = ArchetypeProfile()
    
    # Character archetypes (aggregate over all characters)
    for char_name, char_text in narrative_data['character_texts'].items():
        char_profile = detect_character_archetypes(char_text)
        # Take maximum across characters (presence-based)
        profile.jung_archetypes = np.maximum(
            profile.jung_archetypes,
            char_profile.jung_archetypes
        )
    
    # Plot archetypes
    plot_profile = detect_plot_archetypes(narrative_data['full_text'])
    profile.booker_plots = plot_profile.booker_plots
    profile.campbell_stages = plot_profile.campbell_stages
    profile.polti_situations = plot_profile.polti_situations
    
    # Thematic archetypes
    profile.frye_mythoi = detect_frye_mythoi(narrative_data['full_text'])
    
    # Structural archetypes
    profile.beat_pattern = detect_beat_structure(narrative_data['full_text'])
    profile.vonnegut_shape = detect_story_shape(narrative_data['full_text'])
    
    return profile
